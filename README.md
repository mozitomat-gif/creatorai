# CreatorAI PWA 🎬

Coach algorithmique pour TikTok & Instagram — installable sur Android comme une vraie app.

## 🚀 Déployer sur Vercel (gratuit, 5 min)

### Étape 1 — Prépare les fichiers
Dézippe le dossier `creatorai-pwa` sur ton PC.

### Étape 2 — Crée un compte Vercel
Vas sur https://vercel.com → "Sign Up" → connecte-toi avec GitHub (crée un compte GitHub si besoin, c'est gratuit).

### Étape 3 — Upload le projet
1. Sur Vercel, clique **"Add New Project"**
2. Choisis **"Browse"** pour importer depuis ton PC (pas besoin de GitHub)
   → Ou glisse-dépose le dossier directement
3. Vercel détecte automatiquement que c'est un projet Vite/React
4. Clique **"Deploy"** — attends 1-2 minutes
5. Tu obtiens une URL du type : `https://creatorai-xxxx.vercel.app`

### Étape 4 — Installer sur Android
1. Ouvre l'URL sur Chrome Android
2. Chrome affiche une bannière **"Ajouter à l'écran d'accueil"** → Accepte
3. L'app apparaît sur ton écran d'accueil avec l'icône CreatorAI 🎉
4. Lance-la : elle s'ouvre en plein écran, sans barre de navigateur

## 🔧 Développement local

```bash
npm install
npm run dev
```

## 📦 Build production

```bash
npm run build
```
